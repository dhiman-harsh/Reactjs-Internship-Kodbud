# Task 1: To-Do List Application
Objective
- Learn how to manage dynamic data in React.
What You Need to Do
- Create a To-Do list application where users can:
    - • Add tasks
    - • View tasks
    - • Delete tasks

How to Complete This Task
1. Create an input field to enter tasks.
2. Store tasks using React state.
3. Display tasks using a list component.
4. Add a delete button for each task.

Expected Outcome
A functional To-Do list application built with React.
